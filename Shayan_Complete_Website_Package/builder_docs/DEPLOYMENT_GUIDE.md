# Deployment Guide

## Option A: Static hosting
1. Open the `website/` folder.
2. Replace contact details and legal placeholders.
3. Drag the `website/` folder into Netlify Drop, Vercel, Cloudflare Pages, or upload to cPanel hosting.
4. Test every page and link.
5. Connect custom domain.

## Option B: Webflow / Framer
1. Use the sitemap and content from this package.
2. Rebuild the pages visually in Webflow or Framer.
3. Use each HTML page as copy and layout reference.
4. Connect forms, analytics, SEO, and CMS collections.

## Option C: Next.js / Tailwind
1. Convert each HTML page into a route.
2. Move project content into JSON, MDX, or CMS.
3. Build project detail route template.
4. Add image optimization and SEO schema.
5. Deploy to Vercel.

## Pre-launch QA
- Check mobile menu.
- Check every internal link.
- Test contact form.
- Replace placeholder contact/legal text.
- Add favicon and Open Graph image.
- Run Lighthouse performance/accessibility/SEO checks.
- Add analytics and conversion tracking.
