# Website QA Checklist

## Content
- [ ] Every project has a page
- [ ] No placeholder email remains
- [ ] No placeholder phone remains
- [ ] Legal pages reviewed
- [ ] Contact form works
- [ ] CTAs are clear
- [ ] Spelling checked

## Design
- [ ] Desktop layout clean
- [ ] Tablet layout clean
- [ ] Mobile layout clean
- [ ] Contrast readable
- [ ] Buttons visible
- [ ] Images optimized

## SEO
- [ ] Unique title per page
- [ ] Unique meta description per page
- [ ] Open Graph image added
- [ ] Sitemap generated
- [ ] Robots.txt added
- [ ] Google Search Console connected

## Technical
- [ ] All links work
- [ ] Form backend connected
- [ ] Analytics installed
- [ ] Domain connected
- [ ] SSL active
- [ ] Page speed checked
