# Shayan Complete Website Package

This folder contains a complete multi-page static website starter plus builder handoff documents.

## What is included

- `website/` - deployable static HTML/CSS/JS website
- `builder_docs/` - brief, prompts, content map, deployment notes, QA checklist
- `brand/` - brand direction and style notes
- `Website_Builder_Handoff.docx` - formal builder handoff document
- `Website_Builder_Handoff.pdf` - PDF version of the same handoff document

## Website pages included

Core pages:
- Home
- Projects
- Businesses
- Services
- About
- Contact
- Privacy
- Terms

Project/business pages:
- Agent Command Center (ACC)
- Alphonso Ecosystem
- SessionGuard

- Obsidian Media
- Obsidian Studios
- Blazely
- Founder Project
- Cullinan Construction
- AI Tourist Assistant Platform
- UniUni DSP Opportunity

## How to preview

Open `website/index.html` in a browser.

## How to deploy quickly

Upload the contents of `website/` to Netlify, Vercel static hosting, Cloudflare Pages, GitHub Pages, or any basic hosting account.

## Important before launch

Replace placeholder email, phone number, legal name, privacy policy, terms, screenshots, demo videos, forms, and analytics settings.
